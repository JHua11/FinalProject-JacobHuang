# Final Project

## Author: Jacob Huang 2561210

This is the final project for the course *Introduction to Programming* at Vanier College. This OOP based project aims 
to simulate an organizing system for data of students at an educational establishment. It includes:
 - A Student class 
 - An Address class
 - An Assignment class
 - A Course class
 - A Department class
 - A Util class

and their various methods. The functionalities supported by this project include, among others, registering and dropping students to and from various courses, and generating random scores for students on assignments within a specific course. More details about these classes and the relationships between them can be found in the Class Diagram.
